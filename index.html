This site is Wikipeetia, not Wikipedia

     Wikipeetia The Misspelled Encyclopedia, (not to be confused with Wikipedia The Free Encyclopedia) was a humorous website that contained parody versions of Wikipedia articles where all of the words were misspelled.  This web site is not affiliated with the Wikimedia Foundation in any way.

     The website ran from about 2010 to 2016, but recently I have decided to shut it down and replace it with this simple page because it costs a lot of money to host, and it distracts me from other more useful projects.
